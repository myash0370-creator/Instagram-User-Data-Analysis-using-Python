# 📊 Instagram User Data Analysis (Python)

## 🚀 Overview

This project focuses on extracting and analyzing unstructured Instagram user data using Python. The raw data was cleaned, processed, and converted into a structured format for analysis.

## 🔧 Features

* Parsed raw Instagram profile data into structured JSON format
* Converted textual values (e.g., 2.5K, 1.2M) into numerical values
* Identified users with maximum followers and posts
* Performed category-wise analysis of user profiles
* Handled real-world messy data using custom parsing logic

## 🛠️ Tech Stack

* Python
* JSON
* Basic Data Analysis

## 📈 Key Learnings

* Data Cleaning & Preprocessing
* Handling Unstructured Data
* Writing Custom Parsing Logic
* Extracting Insights from Data

## 📌 Future Improvements

* Use Pandas for advanced data analysis
* Add data visualization (graphs/charts)
* Automate data collection

## 📎 Conclusion

This project demonstrates the ability to transform messy real-world data into meaningful insights using Python.
